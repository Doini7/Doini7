### 🧑‍💻 About Me

<img align="right" alt="Coding" width="280" src="https://raw.githubusercontent.com/Doini7/Doini7/main/developer.gif"/>

I'm a Full-Stack & Mobile Developer with hands-on experience building products from the ground up database design, backend logic, and polished UIs. I care about writing code that scales and creating experiences that feel right to use.

My work spans **Android development with Kotlin**, **cross-platform apps with Flutter**, and solid web foundations across frontend and backend. I like owning the full stack understanding how every layer connects makes me a better developer at each one.

Currently focused on **clean architecture**, **cloud infrastructure**, and **system design**, always looking for ways to build things that last.
